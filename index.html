Welcome to the Stock Intrinsic Value Calculator

This tool is designed to provide an estimate of a company's intrinsic value based on a 10-Year Discounted Free Cash Flow (DFCF) model. It is a web-based replica of the core functionality found in the "KK Prototype Moneycessity" spreadsheet.

How to Use This Calculator

  1. IMPORTANT: Add Your API Key: Before first use, you must get a free API key from Alpha Vantage. Edit this HTML file, find the <script> section at the bottom, and replace 'YOUR_API_KEY_HERE' with your actual key. This is a one-time setup.
  2. Navigate to the "Data Entry" Tab: All inputs required for the calculation are located on this page.
  3. Fetch Company Financials: Enter a stock ticker (e.g., AAPL, MSFT) and click the "Fetch Data" button to automatically populate key financial fields, including the company's Beta and the current Risk-Free Rate.
  4. Provide Your Assumptions: Input your estimates for the company's growth, margins, and other variables.
  5. Calculate & View Results: Once all fields are filled, click the "Calculate & View Dashboard" button. You will automatically be taken to the "Dashboard" tab to see the detailed results.
